# Calculadora

Calculadora Feita em C#, usando o Console.

## 🚀 Executando o Programa

Ao Abrir o Programa, execute o "dotnet build" para buildar o programa e após rode o "dotnet run" para executar o programa.

### 📋 Pré-requisitos

```
Dotnet 6 SDK
Visual Studio Code
```
